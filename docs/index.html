Textts - Text file to sms

Basic app to send txt files by sms (creating sms draft) for MRE platform (including Nokia S30+). If exists automatically open "Texttd.txt" file on storage card or phone memory. RIGHT_SOFTKEY - exit app. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "Texttd.vxp".
